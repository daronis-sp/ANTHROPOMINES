
# Υπολογισμός Ανθρωπομηνών από Excel

## Οδηγίες

1. Εγκαταστήστε τα απαραίτητα πακέτα:
```
pip install streamlit pandas openpyxl python-dateutil
```

2. Τρέξτε την εφαρμογή:
```
streamlit run app.py
```

3. Ανεβάστε αρχείο Excel με περιόδους σε μορφές όπως:
   - 09/2018-09/2019
   - 1/2/2023-15/3/2024

4. Δείτε το αποτέλεσμα και κατεβάστε το αρχείο με τους υπολογισμένους ανθρωπομήνες.
